    Mouse Trap - IOT

    The IOT Mouse Trap project showcases an innovative use of the Orange Pi, integrating various components such as motors, sensors, and buttons to develop a humane and efficient mouse trap. This project represents a convergence of mechanical engineering and information technology, aiming to provide a modern solution to a common household challenge. It demonstrates the practical application of IOT devices in creating smart, responsive systems that interact with the environment in real-time.

    May , 2023

    Objectives

    • Design and construct a humane mouse trap using Orange Pi
    • Incorporate sensors to detect the presence of a mouse
    • Implement motor controls for trap mechanisms
    • Develop user interface elements with buttons for manual control
    • Ensure the device's real-time response and reliability

    Mindful Memory - Task Reminder Web App

    Mindful Memory is a web app crafted to support those with memory impairments, offering a simple interface for managing daily tasks. Utilizing Laravel and Livewire, it provides an effortless and interactive user experience, while Tailwind CSS delivers a clean and responsive design. Our goal is to improve daily living for users, giving them a dependable assistant in their pursuit of autonomy and a better quality of life.

    May 2024

    Objectives

    • Develop a reliable task reminder system for memory-impaired users
    • Utilize Laravel Livewire for a dynamic, single-page application feel
    • Implement a clean, responsive UI with Tailwind CSS
    • Ensure user-friendly experience with intuitive navigation and design
