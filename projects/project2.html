    Noober Carpooling App - Project 4.0

    Project 4.0 - Mobility & Innovation

    Project Overview

    Noober is a smart carpooling application designed to match colleagues for ride-sharing based on personal and logistical preferences such as location, music taste, and event attendance.It was a Project From the company Axxes . The project's duration was September 2024 - February 2025. It leverages predictive logic to suggest optimal carpool combinations and pickup points, promoting sustainability and employee well-being.

    Technologies Used

    Google API
    .NET
    Angular
    K-Means
    AWS
    Kubernetes
    Docker

    Key Features

    • User profile creation (preferences, car type)
    • Predictive carpool logic & ride matching
    • Pickup point optimization
    • RESTful API integration
    • Real-time availability and status
