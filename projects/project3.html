    Tomorrowland Analytics (Epic Data)

    Business Intelligence Project - January 2025

    Project Overview

    Tomorrowland Analytics is a full-stack Business Intelligence project powered by Qlik Sense. Using real-world festival data, we built a dynamic dashboard tailored for stakeholders like the mayor of Boom and the Tomorrowland organizers. The project focuses on transforming raw data into actionable insights through clean data modeling and powerful visual storytelling.

    Technologies Used

    Qlik Sense
    Set Analysis
    Data Modeling
    Qlik Scripting
    Embedded Analytics

    Key Features

    • Qlik Data Model Design
    • Multi-sheet interactive dashboards
    • Advanced data visualizations
    • Set analysis for custom metrics
    • Web storytelling via embedded Qlik objects
